## Usage

Git clone the repo as

```console
git clone https://www.github.com/Utsavjain4561/MapChat
```

## Setup

### Client Side

```console
cd client
npm install
```

### Server Side

```console
cd server
npm install
```

## Startup

### Client Startup

```console
cd client
npm start
```

### Server Startup

```console
cd server
nodemon
```
